核心特性
🔄
学习闭环
Agent 在完成复杂任务后自动创建可复用技能。技能在使用中持续自我改进。FTS5 全文检索跨会话历史。
🧠
持久记忆
跨会话记忆系统 + Honcho 用户建模。Agent 对你的偏好、工作习惯建立持续深化的模型。
⚡
子 Agent 并行
可生成隔离子 Agent 并行处理多个工作流。Python 脚本可通过 RPC 调用工具，零上下文消耗。
⏰
定时自动化
内置 cron 调度器，自然语言定义定时任务，结果推送到任意平台。每日报告、夜间备份、周审计。
🌐
多平台网关
Telegram、Discord、Slack、WhatsApp、Signal——统一单个 gateway 进程。语音备忘录转录，跨平台对话连续性。
☁️
多终端后端
本地、Docker、SSH、Daytona（无服务器）、Modal（按需唤醒）——不绑定你的笔记本，$5 VPS 即可运行。
消息平台支持
📱 Telegram 💬 Discord 💼 Slack 📲 WhatsApp 🔒 Signal 💻 CLI TUI
