Agent 角色详解
🐂
Bull Researcher
多头研究员
构建支持投资的证据链：增长潜力、竞争优势、正面市场信号。利用市场研报、情绪报告、基本面数据形成多头论述，并主动反驳空头观点。
🐻
Bear Researcher
空头研究员
识别投资风险：估值泡沫、竞争威胁、宏观逆风。与多头研究员进行多轮辩论，确保风险充分暴露在决策过程中。
⚖️
Risk Manager
风险管理
三种视角（保守/中性/激进）对多空辩论结果进行风险评估与仲裁，提出风险调整后的投资立场与仓位建议。
📊
Research Manager
研究主管
汇总多空辩论与风险评估，生成结构化投资计划（结构化输出），包括方向、理由、关键风险和仓位建议。
💹
Trader
交易执行
将投资计划转化为具体的交易提案：买/卖/持有 + 数量 + 信心评级。生成结构化 TraderProposal，供 Portfolio Manager 最终确认。
🏦
Portfolio Manager
组合管理
考虑整体仓位、风险敞口和投资组合平衡，对 Trader 提案进行最终批准或调整。输出最终交易决策和持仓更新。
📰
Market Data Agent
数据层
集成 yfinance、新闻 API、社交媒体情绪、基本面数据，为所有分析 Agent 提供统一数据源。支持股票和加密资产。
🧠
Memory System
记忆系统
LangGraph checkpoint 持久化决策历史，支持断点续跑。决策日志记录每次分析结果，用于回测和策略优化。
支持的 LLM 提供商
Claude (Anthropic) GPT (OpenAI) Gemini (Google) Grok (xAI) DeepSeek Qwen / GLM Ollama (本地)
