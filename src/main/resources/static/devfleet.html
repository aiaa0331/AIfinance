Claude DevFleet
Autonomous Coding Agent Platform
多 Agent 自主编码平台。将任务（Mission）分发给多个 Claude Code Agent，每个 Agent 在独立 git worktree 中并行执行，支持子任务分发和依赖调度。基于 Claude Code SDK + MCP。
架构
前端React 19 + Vite:3101
后端FastAPI + SQLite:18801
SDKClaude Code SDK Engine
隔离Git Worktree（每个 Agent 独立）
核心特性
Mission 任务系统：依赖感知的并行任务分发
git worktree 隔离：防止 Agent 间代码冲突
MCP 工具集成：Agent 可调用 Read/Write/Grep/Glob
子 Mission 创建：Agent 可为复杂任务生成子任务
实时状态监控：前端 UI 查看所有 Agent 执行状态
Frontend :3101 / Backend :18801

Case Agent
海关走私案件风险情报系统
持续采集中国海关总署 42 个直属海关的行政处罚案件公示，结构化抽取案件信息，进行风险评分和预测，生成自动化情报报告。
架构
后端FastAPI + Python:8500
数据源海关总署 42 个直属海关
存储SQLite + 结构化数据
调度launchd 定时任务
核心特性
自动采集：覆盖全国 42 个直属海关公示
LLM 抽取：案件要素结构化（货物、处罚金额、违规类型）
风险预测：基于历史案件模式的风险评分
自动脱敏：自然人姓名和身份证号自动处理
定时报告：每周自动生成情报摘要
API :8500
